## Tipe Data dalam Javascript

### #Intro
Terdapat tiga (3) tipe data dasar dalam javascript, yaitu _string_, _integer_, dan _boolean_
Contoh:
```javascript
// string
let nama = 'Erlangga';

// integer
let nilai = 23;

// boolean
let climbing = true;

// array
let acak = ['Erlangga', 23, true];

console.log(nama, nilai, climbing);
console.log(acak);
```
